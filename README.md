# CrewSync

Workforce management made simple.

CrewSync is a workforce scheduling and employee management platform built with ASP.NET Core, helping businesses manage employees, schedules, availability and holiday requests from a single dashboard.

## Features

### Workforce Scheduling

- Create and manage employee shifts
- Weekly schedule dashboard
- Contracted hours tracking
- Overtime warnings

### Employee Management

- Employee profiles
- Role management
- Automatic account creation
- Secure login system

### Holiday Management

- Holiday request workflow
- Manager approval system
- Holiday conflict prevention

### Availability Tracking

- Weekly availability management
- Shift validation
- Scheduling conflict prevention

### Employee Portal

- View upcoming shifts
- Submit holiday requests
- Manage availability
- Change password securely

---

## Technology Stack

### Backend

- ASP.NET Core MVC
- ASP.NET Identity
- Entity Framework Core
- REST API

### Frontend

- Razor Pages
- Bootstrap 5
- Custom UI Design

### Database

- SQL Server

## Getting Started

1. Clone repository

```bash
git clone https://github.com/yourusername/crewsync.git
```

2. Update connection string

3. Run migrations

```bash
Update-Database
```

4. Launch application

```bash
dotnet run
```
